🧑‍💻 Tech Stack
HTML5 – Markup structure

CSS3 & Bootstrap – Styling and responsive layout

JavaScript (Vanilla) – Dynamic UI interactivity

JSON Data Simulation – Mimics API-driven content

GitHub Pages – Deployment (optional)

🔭 Features
🌌 Planet switching via top navigation

📚 3 sections: Overview, Internal Structure, Surface Geology

🪐 Planet image updates dynamically

📈 Stats like Rotation, Revolution, Radius & Temperature

🌈 Modern animated UI with CSS hover effects & 3D planet rotation

📱 Fully responsive across devices

⬆️ Smooth scroll-to-top button
